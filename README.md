## <u>Teamate</u> Project Background
- ### 필요성
  - 다양한 분야에서 문서 형식으로 PDF를 사용
  - 방대한 문서량으로 직접 원하는 정보를 찾아서 사용하는 데 시간이 오래 걸림
    
- ### 기존 해결책의 문제점
  - LLM은 내부 문서를 검색할 수 없으며 보안 문제가 발생할 수 있음

## System Design
   - ### System Requirements
     - PDF 문서 처리 및 텍스트 추출 기능
     - RAG(Retrieval-Augmented Generation) 기반 질의응답 시스템
     - 웹 기반 채팅 인터페이스 및 문서 관리 UI
     - ChromaDB를 활용한 벡터 데이터베이스 및 시맨틱 캐싱
     - Docker 및 Kubernetes를 통한 마이크로서비스 아키텍처 구축
